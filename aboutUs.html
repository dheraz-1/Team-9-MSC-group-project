About the Project

Our team has developed a web application called the Age and Gender Prediction Tool, which utilizes a resnet50 CNN model for predictions. The frontend of our application is designed using HTML, CSS, Bootstrap, and JavaScript to ensure a seamless user experience.


For the backend, we have used Python and Sanic to create a fast and efficient web server. To simplify the process of hosting and maintaining dependencies, we have also incorporated Docker into our application. Additionally, we have leveraged the power of ONNX to package the model in a format that is compatible with server hardware, ensuring optimal performance.


To host our web server, we have chosen a Linux server and have used APIs to establish seamless communication between the frontend and backend. Our team is dedicated to providing a top-quality user experience, and we are committed to delivering the most accurate predictions possible through our innovative Age and Gender Prediction Tool.
